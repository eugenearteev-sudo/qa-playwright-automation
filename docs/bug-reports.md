# Bug Reports

## Bug 1: Incorrect error message on invalid password

**Steps to reproduce:**
1. Open https://the-internet.herokuapp.com/login
2. Enter valid username
3. Enter invalid password
4. Click "Login"

**Expected result:**
A clear and specific error message indicating that the password is incorrect

**Actual result:**
A generic or unclear error message is displayed

**Severity:** Minor  
**Priority:** Medium
